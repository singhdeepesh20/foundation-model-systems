### From Statistical Foundations to Intelligent Learning Architectures

## Overview

This repository is a structured and in-depth exploration of **Machine Learning from first principles to advanced systems design**.

The objective is not just to apply algorithms, but to understand:

* How learning algorithms work mathematically
* Why models succeed or fail
* How to design robust, generalizable systems

This reflects a **top-down + bottom-up approach**:

* Bottom-up → building algorithms from scratch
* Top-down → understanding system-level behavior

---

## Core Learning Flow

```
Mathematics → Classical ML → Optimization → Generalization → Systems Thinking
```

---

## Repository Structure

* **Mathematics** → Linear algebra, probability, statistics
* **Regression Models** → Linear & logistic regression (from scratch)
* **Classification** → KNN, decision trees, ensemble methods
* **Unsupervised Learning** → Clustering, dimensionality reduction
* **Model Evaluation** → Metrics, bias-variance, validation strategies
* **Optimization** → Gradient descent and variants
* **Experiments** → Observations, failures, and improvements

---

## Engineering Approach

### First-Principles Understanding

Every algorithm is implemented from scratch to understand:

* Underlying assumptions
* Mathematical behavior
* Limitations

### Experimental Thinking

Each concept is validated through experiments:

* Hyperparameter sensitivity
* Overfitting vs underfitting
* Data scaling effects

### Comparative Analysis

Models are compared based on:

* Accuracy and generalization
* Computational efficiency
* Interpretability
